# スロットバトル

スロットの出目でダメージを与え合う対戦ゲームです。

## 制作動機

JavaFXをこれまで使用した経験がなかったため、それを使いたくてJavaFXを用いたアニメーション付きのゲームアプリを作成しました。
Swingは授業で使用していたので、JavaFXでどこまで表現できるか試しながら作りました。

## ゲーム概要

スロットを回して出た役に応じて相手にダメージを与え、HPを0にしたら勝利です。
CPU戦・対人戦の両方に対応しており、それぞれ異なる能力を持つキャラクターを5人から選択できます。

## キャラクター

| キャラクター | 特性 |
|---|---|
| 賭太郎 | 補正なし。スタンダードなキャラクター |
| 路地裏の帝王 | 目押しチャンス発生率が高いが子役が出づらい |
| 覚醒するおばちゃん | 子役確率が低めだがBARが出やすい |
| スーパー雀士ガール | 目押しチャンス発生率が低いが、発動時に★図柄が消える |
| 目押しのタツ | 毎ターン目押しチャンスが発生するがリール速度が2倍 |

## 役一覧

| 役 | 条件 | ダメージ |
|---|---|---|
| ７７７ | 3揃い | 30 |
| BAR BAR BAR | 3揃い | 25 |
| ベルベルベル | 3揃い | 15 |
| スイカスイカスイカ | 3揃い | 15 |
| リプレイ | 3揃い | 0（もう1スピン） |
| ★★★ | 3揃い | 0（ゴミ役） |
| チェリー | 1リール目に出るだけ | 5（次スピン7確率アップ） |
| ハズレ | バラバラ | 0 |

## 動作環境

- Java 21
- JavaFX 21

## 実行方法

```
java --module-path "JavaFXのlibフォルダパス" --add-modules javafx.controls,javafx.fxml -jar SlotBattle.jar
```

## 使用技術・工夫した点

- `Canvas` + `AnimationTimer` による縦スクロールリールアニメーション
- `Timeline` / `TranslateTransition` / `FadeTransition` などを使ったBIG WIN演出・ダメージポップアップ
- キャラクターごとに確率やリール速度を変えるパラメータ設計
- 目押しチャンス発生時のリールフリーズ判定
