# スロットバトル

スロットの出目でダメージを与え合う対戦ゲームです。

## 制作動機

JavaFXをこれまで使用した経験がなかったため、それを使いたくてJavaFXを用いたアニメーション付きのゲームアプリを作成しました。
Swingは授業で使用していたので、JavaFXでどこまで表現できるか試しながら作りました。

## ゲーム概要

スロットを回して出た役に応じて相手にダメージを与え、HPを0にしたら勝利です。
役は横3ライン（上段・中段・下段）と斜め2ラインの計5ライン上で判定され、3つ揃った場合のみ成立します。
CPU戦・対人戦の両方に対応しており、それぞれ異なる能力を持つキャラクターを5人から選択できます。

両者のスピンが終わった後にダメージを同時適用するため、両者のHPが同時に0以下になることがあります。
その場合は「エクストラゲーム」に突入し、交互にスピンして先に７７７またはBAR BAR BARを引いた方が勝利となります（両者が同じ結果だった場合は相殺して再戦）。

## キャラクター

| キャラクター | 特性 |
|---|---|
| 賭太郎 | 補正なし。スタンダードなキャラクター |
| 路地裏の帝王 | 目押しチャンス発生率が高いが子役が出づらい |
| 覚醒するおばちゃん | 子役確率が低めだがBARが出やすい |
| スーパー雀士ガール | 目押しチャンス発生率が低いが、発動時に★図柄が消える |
| 目押しのタツ | 毎ターン目押しチャンスが発生するがリール速度が2倍 |

## 役一覧

| 役 | 条件 | ダメージ |
|---|---|---|
| ７７７ | 5ライン中いずれかで3揃い | 30 |
| BAR BAR BAR | 同上 | 25 |
| スイカスイカスイカ | 同上 | 20 |
| ベルベルベル | 同上 | 15 |
| リプレイ | 同上 | 0（もう1スピン、上限3回） |
| ★★★ | 同上 | 0（ゴミ役） |
| チェリー | 左リール中段に出るだけ | 5（次スピン7確率アップ） |
| ハズレ | いずれのラインも揃わない | 0 |

## 目押しチャンス

キャラクターごとに設定された確率で「目押しチャンス」が発生します。
発生時はリールが通常通り回転し、3リールとも自分でSTOPボタンを押して止めます。
止めた盤面をそのまま判定するため、7を狙えばBIG WIN、ベルを狙えばベルベルベルなど、狙った役をそのまま成立させられます。

## 動作環境

- Java 21
- JavaFX 21

## 実行方法

```
java --module-path "JavaFXのlibフォルダパス" --add-modules javafx.controls,javafx.fxml -jar SlotBattle.jar
```

## 使用技術・工夫した点

- `Canvas` + `AnimationTimer` による縦スクロールリールアニメーション
- 画面に表示されている3×3グリッドを実際に走査して役を判定する仕組み（画面外の図柄は判定に含まれない）
- `Timeline` / `TranslateTransition` / `FadeTransition` などを使ったBIG WIN演出・ダメージポップアップ・リール停止時の震動演出
- キャラクターごとに確率やリール速度を変えるパラメータ設計
- 両者のスピン結果を保留してから同時にダメージを適用する処理と、それに伴う「エクストラゲーム」の実装
- 背景画像＋半透明オーバーレイによるカジノ風の世界観演出
